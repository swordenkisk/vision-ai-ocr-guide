# samples/

Place your test images and PDFs here for use with the tutorial notebook and CLI tools.

## Suggested test files

- `invoice.jpg` — a scanned invoice (for standard OCR + Document AI)
- `handwriting.jpg` — a handwritten note or letter
- `form.pdf` — a multi-page PDF form
- `multi_column.jpg` — a newspaper or magazine page (for layout analysis)

These files are intentionally excluded from the repository to avoid storing personal or
proprietary documents in version control.
